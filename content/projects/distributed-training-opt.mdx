# Chess Board with Multi-Agent LLM Integration

- Developed a Pygame-based chess engine from scratch with complete rule support
- Including pawn promotion, castling, and en-passant.
- Integrated a CrewAI-based multi-agent LLM framework enabling LLM vs LLM and LLM vs Human gameplay
- Also added a real-time commentator agent with fault-tolerant response handling.
