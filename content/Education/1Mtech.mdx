# Projects
## Adaptive Edge-Server Collaborative Inference in Consumer Devices
- Built a collaborative inference framework which optimizes the server cost for text generation models with N edge devices
- Profiles latency, network, and compute costs and accordingly adapts the model partitioning strategy
- Utilized the dynamic programming approach to find optimal solution for layers offloading to edge Devices
- Results showed proportionate cost savings with acceptable latency overhead
- Added pipeline parallelim to deal with node failures
